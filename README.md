# SwiftyKit

Make the Swift language easier to use.


## Frameworks included:

- [SwiftyRegex](SwiftyRegex)


## Installation

### Carthage

Add the following to Carfile:

    github "tylingsoft/SwiftyKit"

Run the following command to install:

    carthage update --platform osx
